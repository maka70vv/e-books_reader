<h1>Система озвучивания электронных книг</h1>

Технические требования:
<ul>
<li>
Ubuntu (22.04+)
</li>
<li>
Wayland
</li>
<li>
Python 3.12+
</li>
<li>
Видеокарта Nvidia (опциаонально)
</li>
</ul>

Для запуска проекта:
<br>
pip install -r requirements.txt
<br>
python app.py

Книги для тестирования находятся в директории input
